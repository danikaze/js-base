# Classes specification #

All classes here should implement with different techniques a constructor,
with the following properties/methods:

- Properties:
  - `staticPublicProp`
  - `staticPrivateProp`
  - `publicProp`
  - `privateProp`

- Methods:
  - `staticPublicMethod`
  - `staticPrivilegedMethod`
  - `staticPrivateMethod`
  - `publicMethod`
  - `privilegedMethod`
  - `privateMethod`
